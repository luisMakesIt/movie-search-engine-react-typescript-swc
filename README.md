## Getting started

```bash
git clone https://github.com/luisMakesIt/movie-search-engine-react-typescript-swc.git
cd movie-search-engine-react-typescript-swc
```

### Install dependencies

```bash
npm install
```

### Run locally

```bash
npm run dev
```
